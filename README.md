# Well-Founded-Confidence
#### 자존감이 매칭 성공률에 미치는 영향

## 팀명
`X_train, X_test, X_girlfriend`
<br>

## 프로젝트 설명
캐글의 Speed Dating Experiment 데이터 셋을 이용해 참가자가 참가 전에 자기 자신을 평가한 5가지 특성과,   
매칭 성공 여부를 통해 자존감이 매칭 성공률에 미치는 영향을 파악하고 머신러닝을 통해 예측하는 목표를 가진 프로젝트
<br>

## Members
`공통`: 데이터 전처리, 모델링   
`김강직`: 웹 페이지   
`김민수`: 발표  
`윤규헌`: 모델 성능 평가
